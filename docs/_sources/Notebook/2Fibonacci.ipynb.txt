    "# Algorithm: Fibonacci Sequence Generation\n",
    "\n",
    "## Step 1: Initialize Variables\n",
    "- Set `f1 = 1` (First Fibonacci number).\n",
    "- Set `f2 = 1` (Second Fibonacci number).\n",
    "- Set `n = 1` (Counter variable).\n",
    "\n",
    "## Step 2: Loop until `n` reaches 10\n",
    "- While `n <= 10`, do the following:\n",
    "\n",
    "  1. **Print the current Fibonacci number** (`f1`).\n",
    "  2. **Update Fibonacci values**:\n",
    "     - Assign `f2` to `f1` (shift `f1` to the next Fibonacci number).\n",
    "     - Compute `f1 + f2` and assign it to `f2` (generate the next Fibonacci number).\n",
    "  3. **Increment `n` by 1**.\n",
    "\n",
    "## Step 3: End of Loop\n",
    "- The program stops after printing 10 Fibonacci numbers.\n"
   ]
