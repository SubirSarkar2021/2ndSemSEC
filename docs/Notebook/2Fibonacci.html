Fibonacci numbers

The Fibonacci numbers are the sequence of integers in which each is the sum of the previous two, with the first two numbers being 1, 1. Thus the first few members of the sequence are 1, 1, 2, 3, 5, 8, 13, 21.

Calculate Fibonacci numbers up to 100.

Algorithm: Fibonacci Sequence Generation

Step 1: Initialize Variables

  • Set f1 = 1 (First Fibonacci number).

  • Set f2 = 1 (Second Fibonacci number).

  • Set n = 1 (Counter variable).

Step 2: Loop until n reaches 10

  • While n <= 10, do the following:

    1. Print the current Fibonacci number (f1).

    2. Update Fibonacci values:

      • Assign f2 to f1 (shift f1 to the next Fibonacci number).

      • Compute f1 + f2 and assign it to f2 (generate the next Fibonacci number).

    3. Increment ``n`` by 1.

Step 3: End of Loop

  • The program stops after printing 10 Fibonacci numbers.

```mermaid graph TD; A[Start] –> B[Initialize f1 = 1, f2 = 1, n = 1] B –> C{n <= 10?} C – Yes –> D[Print f1] D –> E[Update f1 = f2] E –> F[Update f2 = f1 + f2] F –> G[Increment n by 1] G –> C C – No –> H[End]

[1]:

f1, f2 = 1, 1
n = 1
while n <=10:
    f1, f2 = f2, f1 + f2
    print(f1)
    n += 1
