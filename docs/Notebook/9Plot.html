import numpy as np
import matplotlib.pyplot as plt

Make a plot of the curve which is defined by

\[x = 2\cos \theta + \cos 2\theta\]
\[y = 2\sin \theta - \sin 2\theta\]

Where \(0 \ge \theta \le 2\pi\) . Take a set of values of \(\theta\) between \(0\) and \(2\pi\), calculate \(x\) and \(y\) for each from the equations above, and then, plot \(y\) as a function \(x\).

Algorithm: Plot a Parametric Curve

Steps:

  1. Define the Range for θ

    • Generate 100 values for ( \theta `) from 0 to ( 2:nbsphinx-math:pi ) using ``numpy.linspace`.

  2. Compute the x and y Coordinates

    • Use the given parametric equations: [ x = 2 \cos `:nbsphinx-math:theta + :nbsphinx-math:cos 2`:nbsphinx-math:theta `] [ y = 2 :nbsphinx-math:sin :nbsphinx-math:theta - :nbsphinx-math:sin 2`:nbsphinx-math:`theta `]

    • Compute the values for ( x ) and ( y ) using numpy.cos() and numpy.sin() functions.

  3. Plot the Curve

    • Use matplotlib.pyplot.plot() to plot ( y ) as a function of ( x ).

    • Set the color of the curve to red and use a dashed linestyle.

  4. Label the Axes

    • Set the x-axis label to “x” and the y-axis label to “y” with a font size of 12.

  5. Set the Title

    • Add the title “Plot of y as a function of x” with a font size of 16.

  6. Display the Plot

    • Use plt.show() to render the plot.

theta = np.linspace(0, 2*np.pi, 100)

x = 2*np.cos(theta) + np.cos(2*theta)
y = 2*np.sin(theta) + np.sin(2*theta)

plt.plot(x, y, color="red", linestyle="dashed")
plt.xlabel("x", fontsize=12)
plt.ylabel("y", fontsize=12)
plt.title("Plot of y as a function x", fontsize=16)
plt.show()
