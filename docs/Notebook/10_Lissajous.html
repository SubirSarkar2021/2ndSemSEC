Lissajous Figure

Consider two simple harmonic motions represented as \begin{align} & y_1 = a_1 \sin\theta\\ & y_2 = a_2 \sin(\theta + \delta) \end{align}

where

  • \(a_1\), \(a_2\) are amplitudes,

  • \(\delta\) is the phase difference which can take some fixed values, and

  • \(\theta\) is a continuous variable from \(0\) to \(2\pi\).

We want to study the relationship between two motions by plotting \(y_2\) against \(y_1\).

Algorithm for Generating Lissajous Figure

Steps:

  1. Define Function ``f(a1, a2, delta)``:

    • Generate 100 values for theta in the range \([0, 2\pi]\).

    • Compute y1 as $ a1 \cdot `:nbsphinx-math:sin`(\theta) $.

    • Compute y2 as $ a2 \cdot `:nbsphinx-math:sin`(\theta `+ :nbsphinx-math:text{radians}`(\delta)) $.

    • Return y1 and y2.

  2. Initialize Variables:

    • Set a1 = 2, a2 = 1, and delta = 90.

    • Call function f(a1, a2, delta) to obtain y1 and y2.

  3. Plot Lissajous Figure:

    • Create a figure of size (5 \times 5).

    • Plot y1 vs. y2 using a red dashed line ('r-').

    • Set x-axis limits to [-2, 2] and y-axis limits to [-2, 2].

    • Label axes with "$y_1$" and "$y_2$", using LaTeX-style notation.

    • Set title as "Lissajous Figure".

    • Add grid and legend.

    • Display the plot.

[44]:

import numpy as np
import matplotlib.pyplot as plt

[45]:

def f(a1, a2, delta):
    theta = np.linspace(0, 2 * np.pi, 100)
    y1 = a1 * np.sin(theta)
    y2 = a2 * np.sin(theta + np.radians(delta))
    return y1, y2

a1, a2, delta = 2, 1, 90
y1, y2 = f(a1, a2, delta)


plt.figure(figsize = (5, 5))
plt.plot(y1, y2, 'r-', label = f"$\\delta$ = {int(delta)}$^\\circ$")
plt.xlim(-2, 2)
plt.ylim(-2, 2)
plt.xlabel("$y_1$", fontsize=14)
plt.ylabel("$y_2$", fontsize=14)
plt.title("Lissajous Figure", fontsize=16)
plt.grid()
plt.legend()
plt.show()
